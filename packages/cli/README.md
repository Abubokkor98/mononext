# Monoframe

Scaffold **production-ready monorepos** in seconds. One command to generate a fully configured Turborepo with Next.js, TypeScript, Tailwind CSS, and more.

## Quick Start

```bash
npx monoframe init
```

That's it. The interactive CLI walks you through everything.

## What You Get

- ⚡ **Turborepo** monorepo with optimized build pipelines
- 🎨 **Next.js** apps with App Router and TypeScript
- 🎯 **shadcn/ui** shared component library with custom presets
- 🖌️ **Tailwind CSS v4** with shared configuration
- 🚀 **Express + TypeScript** backend apps (optional)
- 🎬 **Framer Motion / GSAP / Lenis** animation support (optional)
- ✅ **ESLint + Prettier** or **Biome** for code quality
- 🧪 **Playwright** e2e testing (optional)
- 🔄 **GitHub Actions** CI pipeline (optional)
- 🪝 **Husky + lint-staged** git hooks (optional)

## Usage

### Scaffold a new monorepo

```bash
npx monoframe init
```

### With a project name

```bash
npx monoframe init my-project
```

### Use saved preferences

```bash
npx monoframe init --use-preferences
```

### Reset saved preferences

```bash
npx monoframe init --reset-preferences
```

## Requirements

- **Node.js** 20.9 or later
- **pnpm** 8 or later
- **Git** installed
